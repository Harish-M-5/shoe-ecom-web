# shoe-ecom-web
This is small shoe 🛒E-Commerce web page

---



## 📌 Overview
This project is a **frontend-only** e-commerce shoe website built with React.  
It showcases shoes in a modern responsive grid layout with a simple product detail page.  
The main purpose is to **practice React.js and Tailwind CSS** and to showcase frontend development skills.

---

## ✨ Features
- 🎨 Responsive UI with **Tailwind CSS**
- 👟 Display shoes in **grid view (8 products on home)**
- 🔍 **Search bar** to filter products
- 📄 Product detail page with multiple images
- ⚡ Fast and lightweight (frontend only, no backend)

---

## 🛠️ Technologies Used
- React.js– UI Library
- Tailwind CSS– Styling
- JavaScript (ES6+)
- HTML5

---

## ⚙️ Installation & Setup

1. Clone the repository:
 
 www.https://github.com/shoe-ecommerce.git
cd shoe-ecommerce


3. Install dependencies:

npm install

3. Run the project:

npm start

Open http://localhost:3000 in your browser.
